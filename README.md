# Super Smiles Dental Clinic

A lightweight, mobile-first static website for Super Smiles Dental Clinic in Jammu.

## Local preview

Run any static file server from this directory, for example:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

The project can be deployed directly with GitHub Pages. It has no build step or external runtime dependencies.
